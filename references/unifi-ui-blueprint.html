05
Visual Language — Images, Textures & Effects
No stock photos. Abstract, geometric, trust-building visual system.
🚫 What We DON'T Use
Generic handshake photos · Smiling stock students · Money/cash imagery (feels predatory) · Bright white backgrounds · Generic bank-style illustrations · Purple gradient "fintech" aesthetics
✅ What We DO Use
Abstract geometric mesh backgrounds · Dark glassmorphism cards · Gradient glows (radial, not linear) · Data viz as decoration · Animated SVG icons · Lottie micro-animations for success/loading · University campus skyline silhouettes as subtle texture
🎓
Hero Background
Radial gold-green mesh gradient. Landing page hero. Suggests warmth + growth without showing cash.
💳
Glassmorphism Cards
Frosted glass effect for loan cards and modal overlays. Depth without weight.
Radar / Network Art
Abstract "campus network" — concentric rings suggesting connection and range. Used as background texture on provider screens.
PROCESSING
Verifying UPI...
Loading / States
Skeleton loaders, spinner rings in brand colours, Lottie checkmark on success, shake animation on error.
Earnings Charts
Line charts with gold gradient fill for provider earnings. Green for repayment progress. Minimalist axes, no gridlines.
Admin Bar Charts
Vertical bars for loan volume by month. Tallest bar highlighted in solid gold. No chart borders.
Motion & Animations
✨
Page Load
Staggered fade-up for cards (100ms delay each). Logo animates in from center. Hero text typewriter effect.
🔢
Number Count-Up
All financial amounts animate from 0 to final value on page load. Eased deceleration curve.
⭕
Score Ring
Credit score ring draws itself over 1.2s. Smooth CSS conic-gradient animation.
✅
Success State
Lottie checkmark animation after payment. Green ripple pulse. Confetti burst (subtle, 1 second).
🃏
Card Hover
translateY(-4px) + gold border + glow shadow on loan and stat cards.
🔔
Notification Bell
Gentle swing animation when new notification arrives. Red dot pops in with scale bounce.
06
Layout System
CRED-style single-page feel with section-based navigation
The CRED Layout Philosophy
🖤
Dark is Default
No light mode. The darkness creates premium, focused experience.
📱
Mobile First
Bottom nav bar on mobile. Bottom sheets for modals. Full-screen payment screens.
🔲
Card-Based
Everything lives in cards. No exposed rows or tables in consumer view.
🎯
One Action Per Screen
Each screen has ONE primary CTA. Borrower sees "Borrow." Provider sees "Fund." Admin sees "Review."
Dashboard Layout — Desktop
Collapsed icon sidebar (48px) + main content area + right panel for quick actions. Used for all portal dashboards on desktop.
Mobile Layout
Bottom navigation bar with 4 tabs. Full-bleed content area. Floating action button for primary CTA. No sidebar.
UniFi
💸
Auth / Onboarding — Split Screen
Left: brand visual / illustration. Right: form content. On mobile collapses to full-screen form with top logo.
CONFIRM PAYMENT
Full-Screen Action Screen
Payment confirmation, success/error states, and KYC submission use full-screen focused layouts — no navigation, just the action.

The UniFi Design Principle
Every pixel earns trust. The dark luxury aesthetic signals that this is a serious financial platform — not a toy. Gold communicates aspiration and value. Green communicates growth and safety. Blue commands authority in the admin layer. Together, they create a platform students will want to use, not just have to use.
